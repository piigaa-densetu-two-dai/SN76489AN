～ 夢グルーブの「夢♪」 ～

「夢♪」はMSX用SN76489AN(DCSG)音源ボードです。
ピーガー伝説のsg1000.exe(SG-1000のロムをMSXのロムに変換します)の為に作りました。
音源はIOポート0x3Fと0x7F(書き込み専用、一方はミラー)に接続されています。
5V単一電源動作です。

ピーガー伝説のsg1000.exeはこちら↓
https://github.com/piigaa-densetu-two-dai/sg1000

御注意
SN76489ANは通電時にランダムな音声(ピー、ブー等)を出力します。
ゲームが起動するまで(256in1使用時は256in1メニューが起動するまで)音声が出続けますが不良ではありません。

「夢♪」部品リスト

C1 4.7uF～10uF程度の電解コンデンサ(耐圧6.3V以上・直径5mm)
C2 47uF～100uF程度の電解コンデンサ(耐圧6.3V以上・直径5mm)
C3 0.01uF～0.1uF程度の(積層)セラミックコンデンサ(足間隔5mm)
C4 0.1uF (積層)セラミック又はフィルムコンデンサ(足間隔5mm)
C5 0.01uF～0.1uF程度の(積層)セラミックコンデンサ(足間隔5mm)
C6 0.01uF～0.1uF程度の(積層)セラミックコンデンサ(足間隔5mm)
R1 10Ω 1/4W型抵抗
U1 SN76489AN
